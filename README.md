# SER-DB
This is a repo for showing the demo of SER-DB: subjective database, with an example of food choice.

## Preview the Demo

To view the food exhibit demo locally:

1. Start a local HTTP server in this directory:
   ```bash
   python3 -m http.server 8080
   ```

2. Open your browser and navigate to:
   ```
   http://localhost:8080/food-exhibit-500recipes.html
   ```

3. When done, stop the server with `Ctrl+C`
